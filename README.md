# Graphics

This repo is for graphical things related to unixMiB

## Palette

This is unixMiB official color palette:

- #2DA235
  - RGB: 45, 162, 53
  - CMY: 82, 36, 79
- #424242
  - RGB: 66, 66, 66
  - CMY: 74, 74, 74
- #757575
  - RGB: 117, 117, 117
  - CMY: 54, 54, 54
